Setup Prettier

  • Go to visual studio code and type `ctrl + shift + p`
  • Search for `Preferences: Open Settings (UI)`
  • Search in the search bar `format`
  • Click `Text Editor`
  • Check `Editor: Format On Save`
  • Check `Editor: Format On Paste`
  • Select `Prettier - Code formatter`
